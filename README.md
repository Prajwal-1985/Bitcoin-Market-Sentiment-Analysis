# 📌 Project Overview

Understanding market sentiment is one of the most important aspects of cryptocurrency trading.

This project analyzes the relationship between the **Bitcoin Fear & Greed Index** and **Hyperliquid trader performance** by combining historical trading data with daily sentiment values.

The objective is to discover how different market emotions influence:

- Trading behaviour
- Profitability
- Risk-taking
- Trading volume
- Overall trader performance

The project follows a complete **Data Science workflow**, from data preprocessing to visualization and statistical analysis.

---

# 🎯 Objectives

- Analyze Bitcoin market sentiment using the Fear & Greed Index
- Study trader behaviour under different market conditions
- Merge multiple datasets for unified analysis
- Perform Exploratory Data Analysis (EDA)
- Identify sentiment-driven trading patterns
- Generate business insights for informed trading decisions
